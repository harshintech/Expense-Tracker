## About  
An intuitive expense tracker app to help you manage and monitor your daily expenses with ease. Featuring a sleek, responsive UI designed for seamless user experience across devices.

---

## Features  
- Add and delete expenses  
- Categorize expenses (Food, Shopping, Travel, Bills, etc.)  
- View monthly expense summary  
- Responsive and clean UI  
- Real-time updates on expense entries

---

## Also Included This Features
- User authentication & cloud sync  
- Expense analytics with charts  
- Export/Download data options(XML Sheet)  

---

## Tech Stack  
- React.js for frontend  
- Backend Node.js + Express.js + MongoDB
- Tailwind CSS
- Hosted on Vercel  

---

## How to Use  
1. Open the [live app](https://expense-tracker-plum-psi.vercel.app/)  
2. Add your expense with amount, category, and date  
3. Track and manage your expenses in real-time  
4. Review monthly summaries for better financial control
